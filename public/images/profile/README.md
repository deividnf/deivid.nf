# Foto de perfil da Hero

Coloque sua foto neste caminho para aparecer automaticamente na tag da Hero:

- `public/images/profile/avatar.jpg`

Recomendações:

- Formato: JPG ou PNG
- Tamanho sugerido: 200x200 (quadrada)
- Fundo limpo para melhor leitura no tamanho pequeno

Se quiser usar outro nome/caminho, altere em:

- `lib/site-config.ts` -> `siteConfig.hero.avatarPath`
